# BALIYA — сайт аренды вилл

> Статический лендинг BALIYA (Избербаш, Каспий). 3 страницы, без бэкенда.

## Обзор
Сайт аренды виллового комплекса. 8 домиков от 2 до 6 гостей, 100м до Каспия.
CTA — прямые WhatsApp-ссылки с предзаполненным текстом.

## Стек
- HTML5 + Tailwind CSS (CDN) — без сборки
- Vanilla JS — расписание занятости, FAQ-аккордеон
- Google Fonts: Cormorant Garamond (заголовки) + Inter (текст)
- Без бэкенда: статические файлы, формы — WhatsApp-ссылки

## Архитектура
```
Baliya/
├── index.html      — главная: hero, о комплексе, зоны отдыха
├── cottages.html   — 8 домиков + расписание занятости
├── contacts.html   — контакты, FAQ, карта
├── CLAUDE.md       — этот конфиг
└── Хранилище/      — Obsidian vault проекта
```

## Obsidian Knowledge Vault
Хранилище знаний: `Baliya/Хранилище/`

### При старте сессии
Прочитай `Хранилище/00-home/index.md` и `Хранилище/00-home/текущие приоритеты.md`.

### При завершении (пользователь говорит «сохрани сессию»)
1. Создай заметку в `Хранилище/sessions/` с датой
2. Обнови `Хранилище/00-home/текущие приоритеты.md`
3. Если архитектурное решение — создай заметку в `Хранилище/knowledge/decisions/`
4. Если починили баг — создай заметку в `Хранилище/knowledge/debugging/`

## Команды
- Просмотр: открыть любой `.html` в браузере или `python3 -m http.server` в папке
- Деплой: `vercel --prod` в папке Baliya
