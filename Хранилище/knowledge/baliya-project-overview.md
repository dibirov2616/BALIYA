## Дизайн-система

| Токен | HEX | Назначение |
|---|---|---|
| `wood` | `#C4722A` | бренд-цвет, кнопки, акценты |
| `charcoal` | `#1C1A18` | тёмный фон, навбар |
| `sand` | `#FAF7F2` | светлый фон |
| `green` | `#4A6B3A` | тропический акцент |

- Шрифты: **Cormorant Garamond** (заголовки) + **Inter** (текст)
- Мотив: балийские арочные крыши → CSS border-radius декоры на всех страницах

## Структура сайта

| Файл | Содержание |
|---|---|
| `index.html` | Hero, О комплексе, Лаундж, Барбекю |
| `cottages.html` | 8 карточек домиков + расписание занятости |
| `contacts.html` | WhatsApp, Instagram, адрес, FAQ, карта |

## Домики (8 штук)
Лотос, Бамбук, Пальма, Орхидея, Гибискус, Фикус, Закат, Франжипани.
Вместимость: от 2 до 6 гостей.

## Расписание занятости
- JS-таблица: строки = домики, колонки = месяцы
- Значения: `0` = свободно, `0.5` = частично, `1` = занято
- Google Sheets интеграция: вставить CSV URL в переменную `SHEETS_CSV_URL` в `cottages.html`

## Стек
HTML5 + Tailwind CSS (CDN) + Vanilla JS. Без сборки, без бэкенда.
Один файл на страницу. CTA → прямые WhatsApp-ссылки с предзаполненным текстом.
